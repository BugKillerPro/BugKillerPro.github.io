go自定义json,CustomMarshalJSON

需求描述: golang 的原生 json package 有时会有一些与预期不符合的情况,
例如对接编码(json.Marshal) golang 会默认 “整型浮点数” 如: 1.00 转换为json 的整型 1, 但有时并不希望这种转换.
所以就有了CustomMarshalJSON,即自定义json
